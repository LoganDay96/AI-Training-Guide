Common Mistakes

Learn from others' mistakes so you don't have to make them yourself.

Mistake #1: Using the Wrong Model

The Problem

You start using Code Puppy and it seems... dumb. It gives generic answers, doesn't understand your requests, and you wonder what the hype is about.

The Cause

You're probably using Haiku (the lightweight model) instead of a full model like Claude Opus.

The Fix

Type /model and switch to Claude Opus 4.5 or ChatGPT 5.2. The difference is night and day.

Mistake #2: Being Too Vague

The Problem

You ask "analyze this file" and get a generic, unhelpful response.

The Cause

Vague prompts = vague results. The AI doesn't know what you're actually looking for.

The Fix

Be specific about what you want:

Instead of... Try...
"Analyze this file" "Find the top 10 customers by revenue and show me a breakdown by region"
"Make a chart" "Create a bar chart showing monthly sales trends with blue bars"
"Help me with this" "I need to summarize this 50-page PDF into 5 bullet points for my manager"

Mistake #3: Not Checking the File Path

The Problem

Code Puppy says it can't find your file, or analyzes the wrong thing.

The Cause

You typed the path manually and made a typo, or the file is open in another program.

The Fix

  • Hold Shift, right-click the file, select "Copy as path"
  • Paste that exact path into Code Puppy
  • Make sure Excel/Word isn't currently using the file

Mistake #4: Fighting a Losing Battle

The Problem

You've asked the same thing 10 different ways and it still doesn't work. You're getting frustrated.

The Cause

Sometimes the AI goes down a wrong path and can't recover. The context gets polluted.

The Fix

Follow the Rule of 3:

  1. If it fails 3 times, try a different model
  2. If that fails 3 times, start a completely new conversation
  3. Save a context.md file first so you don't lose progress

Mistake #5: Ignoring Context Limits

The Problem

Code Puppy was working great, but now it's giving generic answers and forgetting what you told it earlier.

The Cause

The conversation has gotten too long. The "context window" (the AI's memory) is full.

The Fix

  1. Ask Code Puppy to save a context.md summary file
  2. Start a new conversation
  3. Tell it to read the context.md to pick up where you left off
Please create a context.md file summarizing what we've built, current state, and next steps.

Mistake #6: Not Using the Interview Technique

The Problem

You have a vague idea of what you want, and the result doesn't match your expectations.

The Cause

You jumped straight to building without clarifying requirements first.

The Fix

Use this magic prompt:

Help me come up with a plan to implement this. DON'T WRITE OR EDIT ANY FILES.
Ask me clarifying questions until you are 95% confident you can complete this task.
Provide options labeled a, b, c, etc. for me to choose from.

Mistake #7: Expecting 0-100 in One Shot

The Problem

You ask for a complete, production-ready application and it doesn't work perfectly.

The Cause

AI is amazing at 0-60 (quick prototypes, analysis, local tools). It struggles with 60-100 (production apps, deployment, live data).

The Fix

Set realistic expectations:

  • Great for: Analysis, dashboards, visualizations, document processing, local tools
  • Harder: Deployed websites, apps with live data, complex integrations

Start small and iterate. Build in chunks, not all at once.

Mistake #8: Not Cleaning Data First

The Problem

Code Puppy misinterprets your spreadsheet or gives weird results.

The Cause

Your data has merged cells, unclear headers, hidden columns, or messy formatting.

The Fix

Before feeding data to Code Puppy:

  • Add clear column headers
  • Remove merged cells
  • Delete unnecessary columns/rows
  • Use consistent formatting (dates, numbers, etc.)
Remember Making mistakes is part of learning! The key is recognizing them quickly and knowing how to recover. Bookmark this page for reference.
