Installing Code Puppy

Getting Code Puppy set up takes less than 5 minutes. Follow these steps to get started.

Prerequisites

  • A Walmart laptop (Windows or Mac)
  • Access to the internet (on VPN or in-office)
  • Windows PowerShell (Windows) or Terminal (Mac)

Installation Steps

  1. Go to the Code Puppy website
    Open your browser and navigate to puppy.walmart.com
  2. Sign in with Walmart Auth
    Click the "Sign in with Walmart Auth" button and authenticate with your Walmart credentials.
  3. Open your terminal
    Windows: Search for "PowerShell" in the Start menu and open it.
    Mac: Open the Terminal app (search in Spotlight or find it in Applications > Utilities).
  4. Copy the installation command
    On the Code Puppy website, find the installation command for your operating system (Windows or Mac) and copy it.
  5. Paste and run the command
    Paste the command into PowerShell/Terminal and press Enter. The installation will begin automatically.
Installation Time The installation typically takes 1-3 minutes depending on your internet connection. You'll see progress messages as it installs.

Running Code Puppy

Once installed, starting Code Puppy is simple:

  1. Open PowerShell (Windows) or Terminal (Mac)
  2. Type code-puppy and press Enter
  3. Start chatting!
    Just type what you want help with in plain English. 
PS C:\Users\YourName> code-puppy
Pro Tip You can run Code Puppy from any directory. If you want to work on files in a specific folder, navigate there first using cd path\to\folder, then run code-puppy.

Updating Code Puppy

Code Puppy updates automatically when you start it. If updates aren't applying:

  1. Close Code Puppy completely
  2. Restart it by typing code-puppy again
  3. If issues persist, reinstall from puppy.walmart.com

Troubleshooting Installation

Problem Solution
Command not recognized Close and reopen PowerShell/Terminal, then try again
Connection errors Make sure you're on VPN or Walmart network
Permission denied Run PowerShell as Administrator (right-click > Run as Administrator)
Installation hangs Check your internet connection; try again in a few minutes
Need Help? If you're still having trouble, reach out in the Code Puppy Teams channel or contact the Code Puppy team directly.
