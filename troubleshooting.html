Common Mistakes

Mistake #1: Using the Wrong Model

Problem: Code Puppy seems dumb and gives generic answers.

Fix: Type /model and switch to Claude Opus 4.5 or ChatGPT 5.2.

Mistake #2: Being Too Vague

Problem: "Analyze this file" gets unhelpful results.

Fix: Be specific: "Find the top 10 customers by revenue and break down by region."

Mistake #3: Wrong File Path

Problem: Code Puppy can't find your file.

Fix: Hold Shift, right-click the file, select "Copy as path", paste that.

Mistake #4: Fighting a Losing Battle

Problem: You've asked the same thing 10 ways and it's not working.

Fix: Rule of 3 - if it fails 3 times, switch models. If still failing, start a new conversation.

Mistake #5: Ignoring Context Limits

Problem: AI was working great, now gives generic answers.

What's happening: The AI has a "context window" - the amount of conversation it can remember at once (like short-term memory). As your chat gets longer with code, files, and back-and-forth, older information gets pushed out. Once full, the AI loses track of earlier decisions and starts giving generic responses.

Auto Compact: Code Puppy has an "auto compact" feature that automatically summarizes older parts of the conversation when the context gets full. This keeps the key points while freeing up space. You'll see a notification when this happens. It helps, but very long sessions may still lose some details.

Fix: Save important decisions to a context.md file, start a fresh chat, and tell the AI to read that file first. Think of it as giving the AI notes to study before starting fresh.

Installation Issues

ProblemSolution
code-puppy not recognized Close and reopen PowerShell, try again
Connection/download errors Make sure you're on the Eagle Profile VPN
Permission denied Right-click PowerShell > Run as Administrator
Installation hangs Check internet connection, try again in a few minutes

If all else fails: Reinstall from puppy.walmart.com

Runtime Issues

File is being used by another process

Close Excel/Word first, or copy the file to a new location.

Model is very slow

  • Complex tasks take longer - be patient
  • Try a different model if speed is critical
  • Break requests into smaller pieces

BigQuery or Confluence agents not working

  • Check you have proper permissions
  • Try restarting Code Puppy
  • Re-authenticate if needed
