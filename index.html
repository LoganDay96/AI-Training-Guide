Get Started

What is Code Puppy?

Code Puppy is Walmart's in-house AI coding assistant, created by Michael Pfaffenberger (Walmart Software Engineer) in May 2025. It's a powerful AI model that can:

  • Write code based on commands in plain English
  • Access Walmart systems including BigQuery, Confluence, and Microsoft 365
  • Analyze documents like Excel spreadsheets and PDFs
  • Help with Walmart-specific research and documentation
Why Code Puppy? Code Puppy is quickly becoming Walmart's go-to AI tool. It's getting powerful very quickly, and the underlying Large Language Models are now good enough for non-technical users to create really useful things.

What is Vibe Coding?

"Vibe Coding" is using natural language (English) to tell an AI agent what you want an app, website, or analysis to do - and then the agent writes the code for you.

  • It's called "vibe" coding because you're just vibing while it does the work
  • You say: "I want to build a dashboard for this data"
  • The agent writes the code for a dashboard
Realistic Expectations 0-60: Incredibly fast. Dashboards, interactive visuals, data analysis, document processing - all great use cases that work well locally on your computer.

60-100: Incredibly slow. Shipping a full website or app, connecting to live data, hosting online - these are more advanced and take more effort.

Good Uses for Code Puppy

Analyze Spreadsheets

Feed an Excel doc into Code Puppy, tell it what analysis you want done, and let it work. Copy and paste the file path to make it quick.

Create Visualizations

Tell Code Puppy to make charts and visuals based on your data. Great for quick reports and presentations.

Document Analysis

Upload PDFs, Word docs, or other files and have Code Puppy summarize, extract, or analyze the content.

Walmart Research

Use the built-in Confluence and BigQuery agents to search internal documentation and query data.
