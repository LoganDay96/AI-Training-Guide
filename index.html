What is Code Puppy?

Code Puppy is Walmart's in-house AI coding assistant, created by Michael Pfaffenberger in May 2025. It can:

  • Write code based on plain English commands
  • Access Walmart systems - BigQuery, Confluence, Microsoft 365
  • Analyze documents - Excel, PDFs, Word docs
  • Research - Search internal documentation

What is Vibe Coding?

Using natural language to tell an AI what you want - and the AI writes the code. You're just vibing while it works.

Realistic Expectations 0-60: Fast. Dashboards, analysis, visualizations, local tools.
60-100: Slow. Production apps, live data, hosting - takes more effort.

Good Uses for Code Puppy

Analyze Spreadsheets

Feed Excel files and get insights, summaries, and analysis.

Create Visualizations

Generate charts and graphs from your data.

Document Analysis

Summarize PDFs, extract data, process documents.

Walmart Research

Search Confluence and query BigQuery directly.
