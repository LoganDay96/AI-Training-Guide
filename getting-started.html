Getting Started with Code Puppy

Now that you have Code Puppy installed, let's learn how to use it effectively.

Choosing the Right Model

Code Puppy supports multiple AI models. Choosing the right one makes a big difference in quality.

Important: Check Your Model Make sure you're using one of the top-tier models. If you're using Haiku (the lightweight model), you might think Code Puppy is not very smart - but it's just using a less capable model!

How to Switch Models

Type /model and press Enter to see your available options.

Recommended models (as of January 2026):

Model Best For
Claude Opus 4.5 Best overall model for most tasks
ChatGPT 5.2 Very good, more methodical, slightly slower
Gemini 3.0 Great for image processing and front-end design

Basic Commands

Code Puppy responds to natural language, but there are also some helpful slash commands:

Command What It Does
/model Show and switch between AI models
/help Display available commands
/clear Clear the current conversation
/exit Exit Code Puppy

Your First Conversation

Just start typing what you want help with. Here are some example prompts to try:

Analyzing Data

I have an Excel file at C:\Users\MyName\Documents\sales_data.xlsx
Please analyze it and tell me the top 5 performing products.

Creating Visualizations

Look at this data file and create a bar chart showing monthly trends.
Save it as a PNG file.

Researching at Walmart

Search Confluence for documentation about the finance reporting process.

Getting Help with Code

Help me write a Python script that reads a CSV file and removes duplicate rows.

Built-in Agents

Code Puppy has special agents for Walmart-specific tasks:

BigQuery Search

Search and query Walmart's BigQuery data warehouse directly from Code Puppy.

Confluence Search

Search Walmart's Confluence documentation to find internal guides and processes.

Agent Creator

Create your own custom agents. Just describe what you want the agent to do, and Code Puppy will set it up for you.

Working with Files

One of Code Puppy's best features is working with files on your computer.

Tip: Use File Paths

Instead of describing where a file is, just paste the full path:

Please analyze C:\Users\MyName\Documents\Q4_Report.xlsx
Quick Path Copy In Windows File Explorer, hold Shift, right-click a file, and select "Copy as path" to get the full file path.

Navigating Folders

Before starting Code Puppy, navigate to your project folder:

cd C:\Users\MyName\Documents\MyProject
code-puppy

Now Code Puppy will see all files in that folder by default.

The Interview Technique

If Code Puppy is confused about what you want, ask it to interview you:

I want to build something but I'm not sure exactly what.
Interview me until you understand what I need.
Ask me numbered questions with multiple choice options (a, b, c, etc.)

This is a great way to clarify complex requests!

The Best Prompt for Complex Tasks

Use this prompt to get Code Puppy to really understand what you want:

"Help me come up with a plan to implement this. DON'T WRITE OR EDIT ANY FILES. Ask me clarifying questions until you are 95% confident you can complete this task successfully. Provide options labeled a, b, c, etc. for me to choose from."
