Understanding the Models

Current Recommended Models (January 2026)

Model Provider Notes
Claude Opus 4.5 Anthropic Best overall for most tasks
ChatGPT 5.2 OpenAI Very good, slightly slower, methodical
Gemini 3.0 Google Great for images and front-end design
Model Tips
  • Claude tends to write detailed documentation and README files
  • ChatGPT is often better at understanding complex directions
  • Gemini excels at visual tasks and image processing
Don't be afraid to switch between models when one gets stuck!

Helpful Terminology

Term Definition
CLI Command Line Interface - interact with your computer using text commands
GUI Graphical User Interface - the visual windows, buttons, and menus you click on
IDE Integrated Development Environment - software for writing, testing, and debugging code
Repository (Repo) A project folder containing your code, tracked by Git
Branch A version of your code - useful for experimenting without breaking the main version
Commit A saved snapshot of your code changes
Context Window The amount of conversation an AI can "remember" at once
Token A unit of text (roughly 4 characters or 3/4 of a word) - used to measure context size

Tools for the Next Level

VS Code + GitHub Copilot

If you want to go beyond Code Puppy into full development:

  1. Install VS Code
  2. Request GitHub Copilot access via MyTech
  3. Install the GitHub Copilot extension in VS Code
Start with Code Puppy VS Code is more powerful but also more complex. We recommend getting comfortable with Code Puppy first before moving to VS Code for larger projects.

GitHub Desktop

For managing your code versions:

  • Download: GitHub Desktop
  • Great for undoing mistakes and tracking changes
  • Connects your local projects to GitHub

Common Coding Languages

As you progress, you might encounter these:

Language What It's For
Python General programming, data analysis, AI/ML, automation
SQL Querying and managing databases
HTML Structure of web pages
CSS Styling and layout of web pages
JavaScript Making web pages interactive
TypeScript JavaScript with better error checking
You Don't Need to Learn These With Code Puppy and AI agents, you can accomplish a lot without learning to code yourself. The AI writes the code - you just need to describe what you want. However, understanding the basics can help you communicate more effectively with the AI.

The Best Prompt

When starting a complex project:

Help me come up with a plan to implement this.
DON'T WRITE OR EDIT ANY FILES.
Ask me clarifying questions until you are 95% confident
you can complete this task successfully.
Provide options labeled a, b, c, etc. for me to choose from.

When Things Go Wrong

  1. Check your model (/model)
  2. Try a different model
  3. Start a new conversation
  4. Ask Code Puppy to create a handoff file (e.g., context.md) before starting fresh
