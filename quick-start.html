Get up and running with Code Puppy in under 5 minutes. Just the essentials.

TL;DR Install from puppy.walmart.com, run code-puppy in PowerShell, type /model to pick Claude Opus 4.5, start chatting.

Step 1: Install

  1. Go to puppy.walmart.com
  2. Sign in with Walmart Auth
  3. Open PowerShell (search "PowerShell" in Start menu)
  4. Copy the install command from the website, paste it, hit Enter

Step 2: Run

Open PowerShell and type:

code-puppy

Step 3: Pick a Good Model

Type this command first:

/model

Select Claude Opus 4.5 (best overall) or ChatGPT 5.2.

Don't Skip This! If you're on Haiku (the default lightweight model), Code Puppy will seem dumb. Always switch to a full model.

Step 4: Start Chatting

Just type what you want in plain English. Try these:

Analyze a file:

Please analyze C:\Users\YourName\Documents\data.xlsx and summarize the key findings

Search Walmart docs:

Search Confluence for documentation about expense reporting

Create a chart:

Look at this Excel file and create a bar chart of sales by month

Get help with a complex task:

I want to build something but I'm not sure what. Interview me until you understand what I need.

Terminal Shortcuts

Shortcut What It Does
Ctrl + C Stop Code Puppy from thinking (cancel current request)
Up / Down arrows Scroll through your past prompts
/ Show all available slash commands

Essential Commands

Command What It Does
/model Switch AI models
/agent Switch between agents (BigQuery, Confluence, etc.)
/resume Resume a previous conversation
/help Show all commands
/clear Clear conversation
/exit Exit Code Puppy

Pro Tips

  • Paste file paths directly - Hold Shift, right-click a file, "Copy as path"
  • Be specific - "Analyze sales" is vague; "Find the top 5 products by revenue" is better
  • Screenshot errors - Paste them right into the chat
  • If it gets confused - Start a new conversation
