üí° Quick Tips

  • Be specific: "Find top 10 customers by revenue" not "analyze this"
  • Rule of 3: If it fails 3 times, switch models
  • Copy file paths: Shift + right-click ‚Üí "Copy as path"
  • Start fresh: When context is full, save a context.md and start new chat
  • Save often: Backup working folders with dated names
